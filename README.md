# vue-typescript-weather
在學習 vue 整合 typeScript 的路上，自行實作的查詢天氣小工具。

## 功能：
1. 查詢天氣
由於 API 是用國外的 [OpenWeatherMap](https://openweathermap.org/current)，故只能搜尋英文。
![](https://i.imgur.com/pGxPipE.png)

2. 歷史記錄
記錄最新搜尋前十筆
![](https://i.imgur.com/UzKsaJR.png)


## 技術
vue / vuex / typeScript
